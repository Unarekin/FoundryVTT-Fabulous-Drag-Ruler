# Fabulous Drag Ruler
Fabulous Drag Ruler adds some customizability to the token drag ruler added in FoundryVTT v13.  It allows you to enable or disable the ruler by default for gridded or gridless scenes, toggle it on or off per-scene, and to customize the color and size a bit.


# Installation

## Semi-Automatic Installation
You can install this module by copyin and pasting the following manifest URL in the text field at the bottom of the Foundry module installer:
```
https://github.com/Unarekin/FoundryVTT-Fabulous-Drag-Ruler/releases/latest/download/module.json
```

# Usage
Fabulous Drag Ruler is simple to use, and is not overburdened with a great many configuration options.

## Game Settings

In the Game Settings window, the drag ruler can be enabled or disabled by default for gridless or gridded scenes.  In this same window, you can access the override settings for each portion of the drag ruler.
